Ternary operator in Javascript

It takes three operands: a ? 'true' : 'false'

Same as this if statement:

```
if (a) { // check if the condition is truthy
    // return true
} else {
    // return false
}
```

Common usages:

- inside react return
- return object value
- inline condition operator
